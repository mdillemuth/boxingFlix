fightFlix API Documentation

Yo Adrian! Find all of your favorite combat films in one place,
easily accessible through a modern RESTful API.

About

This website provides a RESTful API interface to return JSON-formatted data related all films fighting.

Features

  • Return a list of all fighting movies to the user
  • Return data (description, genre, director, and image URL about a single movie to the user
  • Return data about a genre (description) by name/title
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update user info (username, password, email, date of birth)
  • Allow users to add or remove a movie to their list of favorites
  • Allow users to deregister
Accessing Movie Database
HTTP Method URL Logic Request Format Response Format
GET /api/movies Return a list of all movies N/A Array containing JSON objects with data of all movies
GET /api/movies/{Title} Return data about a single movie N/A JSON object with data of a single movie
GET /api/movies/genres/{Genre} Return all movies containing a specific genre N/A Array of JSON objects with data of all movies that contain specified genre
GET /api/movies/directors/{Director} Return data about a single director N/A JSON object with data of a single director
Accessing Users Database
HTTP Method URL Logic Request Format Response Format
POST /api/users Add a new user JSON JSON object with data of added user
PUT /api/users/{Username} Update data of a single user JSON JSON object with updated data of user
POST /api/users/{Username}/favorites/{Title} Add a movie to user's list of favorites N/A An array of the updated list of the user's favorite movies.
DELETE /api/users/{Username}/favorites/{Title} Remove single movie from user's list of favorites N/A An array of the updated list of the user's favorite movies.
DELETE /api/users/{Username} Remove a user's account N/A A text message confirming that the user's account was successfully removed

Sample Requests & Responses for /api/movies

GET request for all movies. (e.g. /api/movies)

GET request for a single movies. (e.g. /api/movies/Rocky%20II)

GET request for a specific movie genre (e.g. /api/movies/genres/Boxing)
